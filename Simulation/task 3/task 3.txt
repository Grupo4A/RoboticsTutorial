Function main
inicio:
Motor On
Power High
Speed 30
Accel 30, 30
SpeedS 500
AccelS 5000
Go P24 /15
Wait Sw(0) = Off
Do While Sw(0) = Off And Sw(1) = Off
	If Sw(2) = On Then
		GoTo random
	EndIf
Loop
If Sw(0) = On Then
Xqt task3
Do While Sw(0) = On And Sw(7) = Off
	If Sw(4) = On Then
    	Halt task3
  	Else
		Resume task3
  	EndIf
Loop
Quit task3
Off 8
Wait 5
Go P24 /15
GoTo inicio
EndIf
reverse
GoTo inicio
random:
random1
GoTo inicio
Fend

Function task3
bs = 0
ts = 0
blockheight = 6
tokenheight = 6


Tool 1
Go P24 /15
check

For c = 0 To cantblock - 1
	If block(0) = 0 Or block(1) = 0 Or block(2) = 0 Then
		Pickcu3()
		Placecufix3()
		Placetraycu3()
		bs = bs + 1
		Print bs
	EndIf
Next c
For c = 0 To canttoken - 1
	If token(0) = 0 Or token(1) = 0 Or token(2) = 0 Then
		Pickcir3()
		Placecircufix3()
		Placetraycir3()
		ts = ts + 1
		Print ts
	EndIf
Next c
For c = 0 To 10
		If token(0) = 1 Or token(1) = 1 Or token(2) = 1 Then
		Picktraycir3()
		Placecir3()
		EndIf
Next c
For c = 0 To 10
		If block(0) = 1 Or block(1) = 1 Or block(2) = 1 Then
		Picktraycu3()
		Placecu3()
		EndIf
Next c
Go P24 /15
Fend
